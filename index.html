Las tecnologías que usa la WEB

  • ¿Preparados?

HTML

Hypertext Markup Language

HTML es el lenguaje de marcado estándar para crear páginas web. Utiliza una estructura jerárquica de etiquetas para organizar y presentar contenido, como texto, imágenes, enlaces, formularios, etc. Uso: Define la estructura básica y el contenido de una página web, permitiendo la creación de enlaces, párrafos, encabezados, listas, y más.

CSS

Cascading Styling Sheet

CSS es un lenguaje de estilo que define cómo se presenta y se visualiza el contenido HTML. Permite aplicar estilos como colores, fuentes, márgenes, y posicionamiento a elementos HTML. Uso: Mejora la apariencia y el diseño de una página web, facilitando la separación entre la estructura (HTML) y el estilo visual (CSS).

Y finalmente...

JavaScript

JavaScript es un lenguaje de programación de alto nivel que se utiliza para agregar interactividad y dinamismo a las páginas web. Puede manipular el contenido, responder a eventos del usuario y comunicarse con el servidor. Uso: Permite crear experiencias interactivas en el lado del cliente, validar formularios, realizar animaciones, y actualizar dinámicamente el contenido de una página web.

Construcción de la Página

Esta página web fue construida utilizando tecnologías web estándar, incluyendo HTML, CSS y JavaScript. Aquí hay una breve descripción de cada componente:

HTML

El HTML proporciona la estructura básica de la página. Utilizamos etiquetas HTML para organizar y presentar el contenido, como encabezados, párrafos, enlaces y más.

        
            <html>
                <head>
                    <title>Título de la Página</title>
                </head>
                <body>
                    <h1>Mi Página Web</h1>
                    <p>¡Hola, mundo!</p>
                    <a href="#seccion1">Enlace a Sección 1</a>
                </body>
            </html>

CSS

El CSS se utiliza para dar estilo a la página. Define la apariencia visual, los colores, las fuentes y el diseño general. Facilita la separación entre la estructura (HTML) y el estilo visual.

        
            body {
                font-family: 'Arial', sans-serif;
                background-color: #f0f0f0;
                margin: 0;
                padding: 0;
            }

            h1 {
                color: #333;
                text-align: center;
            }

            .container {
                width: 80%;
                margin: 0 auto;
            }

JavaScript

Pasos Lógicos para Crear y Desplegar el Proyecto

1. Creación del Proyecto

Comencé creando un nuevo directorio en mis documentos para el proyecto. Dentro de este directorio, creé archivos y carpetas necesarios para la estructura del proyecto.

2. Configuración del Entorno de Desarrollo

Abrí Visual Studio Code para desarrollar el proyecto. Organizé los archivos para seguir el principio de "Separation of Concerns", colocando HTML, CSS y JavaScript en carpetas separadas.

3. Diseño y Estilo

Utilicé imágenes para los logos, que fueron colocadas en la carpeta de imágenes. El CSS y SCSS se utilizaron para dar estilo a la página, mejorando la apariencia y el diseño.

4. Desarrollo de la Funcionalidad

Para la funcionalidad interactiva, escribí código JavaScript. Las funciones JavaScript se utilizaron para cambiar el color de fondo.

5. Conexión entre Archivos

Enlazé los archivos HTML, CSS y JavaScript para asegurar una integración adecuada. El archivo JavaScript se vincula al HTML para que las funciones puedan ser invocadas desde los elementos de la página.

6. Control de Versiones y GitHub

Subí el proyecto a GitHub para el control de versiones.

7. Despliegue en GitHub Pages

Finalmente, desplegué el proyecto en GitHub Pages. Esto permite que la página sea accesible en línea para que otros puedan verla y utilizarla.

Estos son pasos generales y simplificados. Puedes revisar el repositorio en GitHub para obtener detalles más específicos y explorar el código fuente.
